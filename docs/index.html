Kyde

A fast native Git client and diff editor that opens instantly.

Side-by-side diffs · hunk staging · IDE-grade commit view · GPU-rendered · written in Rust

Download View on GitHub

Built for one workflow, done fast

The commit and diff view you live in — without waiting for a JVM to start.

⚡

Instant & native

Native GPU rendering on gpui — no web, no Electron. ~120fps scrolling a 37k-line file.

⇄

Visual hunk staging

Side-by-side diff with word-level highlighting and a center gutter to stage or revert hunks — git add -p, made visual.

🌳

Browse & edit

Open a folder, edit with tree-sitter syntax highlighting, opt-in language packs, code folding and find/replace.

⎇

Branches & rollback

Searchable branch switcher, commit, push, and a native rollback window with a checkbox tree of changes.

🎨

Familiar dark theme

A hand-tuned dark palette that feels at home to anyone who's lived in a modern IDE. Configurable at runtime.

⌨️

Your keymap

WebStorm or VSCode presets with per-action overrides. Fuzzy Go to File and Find Action.

Install

macOS Linux Windows 
# macOS — download kyde-macos.zip from Releases, then:
xattr -dr com.apple.quarantine /Applications/Kyde.app

# or build from source
cargo build --release
